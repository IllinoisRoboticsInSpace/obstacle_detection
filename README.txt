====INSTALLATION====
1. Remove all libfreenect and freenect packages from the computer through the software center.
2. Remove remaining files from the computer that start with "libfreenect.so" or "libfreenect.h" or "libfreenect.hpp"
3. Install ONLY the package "libfreenect-dev" from the software center.
4. You should now be able to build and run the code using "make" while in the obstacle_detection folder.

====BUILDING AND RUNNING====
1. "cd" into obstacle_detection folder.
2. Type "make" or "make all" to build all projects, or just type "make <target>" to build a specific program.
3. Read the Makefile. It is simple and you will learn something.
